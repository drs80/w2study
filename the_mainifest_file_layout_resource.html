Android Manifest File and Layout Resources

Android Manifest File (AndroidManifest.xml)

Manifest file किसी भी Android app का मुख्य Configuration File होती है। यह फ़ाइल App Components, Permissions, Hardware Features, और App Metadata को define करती है।

Manifest File के मुख्य Sections

  • <manifest> → यह tag पूरे Manifest file को wrap करता है और package name define करता है।
  • <application> → App की settings, icons, themes, और components (activities, services, receivers) को define करता है।
  • <activity> → App में मौजूद Activities को define करता है।
  • <uses-permission> → App को जिन Permissions की ज़रूरत होती है, उन्हें specify करता है (जैसे internet access, camera, location, आदि)।
  • <uses-sdk> → App के लिए Minimum और Target SDK Level सेट करता है।

Example of AndroidManifest.xml

<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.myapp">

    <uses-permission android:name="android.permission.INTERNET" />
    <uses-permission android:name="android.permission.CAMERA" />

    <application
        android:allowBackup="true"
        android:icon="@mipmap/ic_launcher"
        android:label="MyApp"
        android:theme="@style/Theme.MyApp">

        <activity android:name=".MainActivity">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />
                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
        </activity>

    </application>

</manifest>

Android Layout Resources (res/layout/)

Layout Resources वह XML files होती हैं, जो app के UI (User Interface) को define करती हैं। ये files res/layout/ folder में होती हैं।

मुख्य Layout Types

  • LinearLayout → Views को एक line में (horizontal/vertical) व्यवस्थित करता है।
  • RelativeLayout → Views को एक-दूसरे के relative में position करने की अनुमति देता है।
  • ConstraintLayout → Modern Layout, जो complex UI Design के लिए इस्तेमाल होता है।
  • FrameLayout → एक simple container, जो single view को hold करता है।
  • TableLayout → Data को rows और columns में show करता है।

Example of activity_main.xml (UI Layout File)

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello, World!"
        android:textSize="20sp"
        android:textColor="#000000"/>

    <Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Click Me" />
</LinearLayout>

Manifest और Layout में Connection

AndroidManifest.xml में जो activity define की गई है, वह layout XML file से connected होती है।

Example:

setContentView(R.layout.activity_main);

इसका मतलब activity_main.xml को UI के लिए load करना।
